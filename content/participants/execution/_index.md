## このセクションについて

**実行** では、ローカルでの **サーバ起動** と **エージェント起動** を完了させるための手順をまとめます。
準備が整っていれば、ここを 1 → 2 の順に進めるだけで対戦が始められます。

---

## 目次

1. **[サーバ起動方法](./server.md)**
   OS別（Linux / Windows / macOS Intel / Apple Silicon）の起動コマンド、   既定の接続先（`ws://127.0.0.1:8080/ws`）、ログ保存先の考え方を解説。

2. **[エージェント起動方法](./agent.md)**
   仮想環境の有効化、`config.yml` の **接続先URL / 起動数 / チーム名** の確認、
   `python src/main.py`（または `-c` オプション）での起動方法を説明。
